# Texas Hold'em Unity Project

本仓库为一个用于教学示例的德州扑克实现（Unity 2022.x 项目）。

- 术语表（中文/英文）位于：Assets/Docs/Glossary.md

## 快速开始
- 在 Unity 中打开本项目并运行 SampleScene 或使用 `Tools -> Setup Poker Scene` 创建最小界面。
- 术语表包含对 `Card`、`Deck`、`Draw`、`Flop`、`Turn`、`River`、`Pot` 等常见术语的中文解释，便于阅读源码与调试。
